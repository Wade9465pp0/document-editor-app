# Document Editor App

## Features
- Create, edit, save documents
- Rich text editor
- File upload (.txt)
- Basic sharing

## Tech Stack
- React (Vite)
- FastAPI
- SQLite

## Live Demo
Frontend: https://document-editor-app.vercel.app  
Backend: https://document-editor-app-1-ctnr.onrender.com  

## Run Locally

Backend:
cd backend
uvicorn main:app --reload

Frontend:
cd frontend
npm install
npm run dev
